# PiggyEat
Can think of what to eat today, but got a list of food in mind?
This app randomly suggests a choice of food from the stored list entered by the user.

## Requirement:
* ios 13.0 or above

## Technology Used:
* Coredata for persisting data
* Cocoapod
* Storyboard
